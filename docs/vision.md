# ReddWall Vision

## Problem

Moderators often need to contain a risky thread before they have time to clean
it comment by comment. Spam bursts, brigading, heated dogpiles, and repeated
rule-breaking replies can spread through a post while a mod is still opening
tabs and checking context.

## Solution

ReddWall is a Devvit moderation tool that lets a moderator preview and contain
either a selected comment thread or an entire post from native Reddit menus.
The moderator can lock comments, remove comments, filter the operation to
specific phrases, protect distinguished mod and admin comments, lock the post,
and add strict crowd control from one form.

## Impact

ReddWall reduces the time between "this thread is going bad" and "the thread is
contained." It keeps moderators in Reddit, makes the blast radius visible before
changes are applied, and records recent previews and runs so a team can see what
was attempted.

## Safety Bar

- Preview-first defaults make the target count visible before cleanup.
- Lock and remove are separate controls.
- Distinguished comments are protected by default.
- Phrase targeting narrows actions when only one spam pattern needs cleanup.
- Recent audit history records the action, options, target count, skips, and
  partial action failures.
